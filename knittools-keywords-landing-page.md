# KnitTools Landing Page — Keyword Guide

Purpose: guide for building the landing page so that meta tags, headings, and body text use the right keywords naturally.

---

## Meta Tags

```html
<title>KnitTools — The Complete Knitting Toolkit App for Android</title>
<meta name="description" content="Row counter, gauge calculator, yarn estimator, cast on calculator, needle size chart & project tracker. All-in-one knitting app for Android. No ads, no tracking, works offline.">
```

- Title contains brand name + primary keyword ("knitting toolkit")
- Description lists all tools naturally — works for both Google indexing and click-through rate
- Keep title under 60 characters, description under 160

---

## Page Structure

### H1 (one per page)
Must contain the primary keyword. Examples:

- "The Complete Knitting Toolkit for Android"
- "Every Knitting Tool You Need, In One App"

The word "knitting toolkit" should appear in the H1 in some natural form.

### H2 Sections
Each H2 maps to one tool or feature and targets its own keyword. Suggested structure:

| H2 Heading | Target Keyword |
|---|---|
| Smart Row Counter with Voice Commands | knitting row counter |
| Gauge Calculator — Match Any Pattern | knitting gauge calculator |
| Cast On Calculator — Get the Right Stitch Count | cast on calculator |
| Yarn Estimator — Never Run Out Mid-Project | yarn estimator |
| Needle Size Converter — US, UK & Metric | needle size converter |
| Track Every Knitting Project | knitting project tracker |
| Scan Yarn Labels with Your Camera | yarn label scanner |
| Search Ravelry Patterns | ravelry pattern search |
| 100% Offline, Zero Tracking, No Ads | offline knitting app, privacy, no ads |

## Body Text Keywords Per Section

Under each H2, write 2–4 sentences describing the feature. Use secondary keywords from the same cluster naturally. Do NOT keyword-stuff — if a word doesn't fit naturally, leave it out.

### Row Counter section
Primary: knitting row counter, row counter app
Secondary: stitch counter, track rows, reminders, voice commands, hands-free, undo

### Gauge Calculator section
Primary: knitting gauge calculator, gauge converter
Secondary: swatch, stitches per inch, convert gauge, pattern gauge, adjust pattern

### Cast On Calculator section
Primary: cast on calculator
Secondary: how many stitches, cast on for width, pattern repeat, edge stitches

### Yarn Estimator section
Primary: yarn estimator, yarn calculator
Secondary: how much yarn, skeins needed, yardage, yarn quantity

### Needle Size Converter section
Primary: needle size converter, needle size chart
Secondary: US to metric, UK sizes, knitting needle conversion

### Project Tracker section
Primary: knitting project tracker
Secondary: organize projects, track progress, knitting time, session tracking, WIP

### Yarn Label Scanner section
Primary: yarn label scanner, scan yarn label
Secondary: OCR, camera, yarn details, fiber content, weight category

### Ravelry section
Primary: ravelry pattern search
Secondary: find patterns, save patterns, start project from pattern

### Privacy / Offline section
Primary: offline knitting app, no ads knitting app, privacy
Secondary: no tracking, no data collection, works without internet, one-time purchase

## General Principles

1. **One H1, many H2s** — the H1 is the page's main keyword signal; H2s cover individual tools
2. **Natural language** — Google uses NLP to understand semantic meaning, so write for humans first
3. **Don't repeat keywords excessively** — using "knitting" in every single sentence hurts readability and Google can penalize it
4. **Link to blog articles** — each tool section can link to its corresponding blog article (e.g., gauge section links to "How to Calculate Knitting Gauge" article). This passes SEO authority between pages
5. **CTA to Play Store** — include a clear download link/button. When the app is live, the backlink from landing page to Play Store listing directly boosts Play Store ranking
6. **Alt text on images** — every screenshot or illustration should have descriptive alt text that includes a relevant keyword (e.g., `alt="KnitTools gauge calculator screen showing swatch measurement"`)
